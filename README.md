# Glass Morphism Draggable Card Demo

Simple demo showing Glass Morphism UI.

## Screenshot

![Demo gif of UI](https://github.com/gabalicious/glass-morphism/raw/main/screenshot.gif)

## Getting Started

### Dependencies

* parcel
* sass
* nodejs

### Installing

```
npm i
```

### Executing program

```
npm start
```
